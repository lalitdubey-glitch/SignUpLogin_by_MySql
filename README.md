# 🔐 SignUpLogin — ASP.NET Core MVC + MySQL

A full authentication system built with ASP.NET Core MVC and MySQL.

## 🚀 Features
- **User Signup & Login** with session-based authentication
- **BCrypt Password Hashing** — secure password storage
- **Role-Based Authorization** with `[Authorize]`
- **RestSharp** for external API calls
- **MySQL** as database with `MySql.Data` connector
- Full **CRUD** operations

## 🏗️ Tech Stack
| Layer | Technology |
|---|---|
| Framework | ASP.NET Core MVC |
| Language | C# |
| Database | MySQL |
| Auth | BCrypt.Net + Session |
| HTTP Client | RestSharp |
| JSON | Newtonsoft.Json |
